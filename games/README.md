# games

Small playable games.

## Projects

- `quiz/` — OOP true/false quiz game. `QuizBrain` drives question sequencing and scoring; question data sourced from Open Trivia DB JSON.
- `snake/` — classic Snake with OOP architecture (`Snake`, `Food`, `Scoreboard` classes), difficulty selection, and a toxic scoreboard for low scores.
- `pong/` — two-player Pong. Ball speed ramps on each paddle hit; game ends after three out-of-bounds events.
- `turtle-crossing/` — Frogger-style crossing game. Car spawn rate is random; speed increases with each successful crossing.
- `states-quiz/` — US states geography quiz using pandas to read coordinate data and turtle to label each correct guess on a map.
