# fundamentals

Basic Python syntax, control flow, and early project work.

## Scripts

- `band_name_generator.py` — string input/output, variable concatenation
- `tip_calculator.py` — float arithmetic, f-strings
- `pizza_calculator.py` — nested conditionals, while-loop input validation
- `treasure_island.py` — branching text adventure with ASCII art
- `rock_paper_scissors.py` — random module, list indexing
- `password_generator.py` — random.choice, list shuffle
- `hangman.py` — string iteration, list membership, lives counter (depends on missing `hangman_words.py` / `hangman_art.py`)
- `caesar_cipher.py` — alphabet indexing, modulo wrap, encode/decode function
- `blind_auction.py` — dictionary accumulation, highest-value search
- `calculator.py` — functions as dict values, recursive loop
- `blackjack.py` — card game logic, ace-handling edge case
- `higher_lower.py` — data-driven comparison game (depends on missing `game_data.py` / `art.py`)
- `coffee_machine.py` — procedural coffee machine simulation (capstone)
- `coffee_machine_v1.py` — earlier iteration of the coffee machine, less error handling
