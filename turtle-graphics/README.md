# turtle-graphics

Drawing exercises using Python's `turtle` module.

## Programs

- `spirograph/spirograph.py` — draws a multi-color spirograph by rotating a circle in small angle increments with random RGB colors
- `color-palette/color_extractor.py` — uses `colorgram` to pull RGB values from a reference image (`hurst-spot-painting.jpg`)
- `color-palette/color_palette.py` — recreates a Damien Hirst-style dot grid painting using the extracted palette
- `etch-a-sketch/etch_a_sketch.py` — WASD-controlled turtle drawing with a clear/reset key
- `turtle-race/turtle_race.py` — six turtles race across the screen; player bets on a color before the race starts
