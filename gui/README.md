# gui

Tkinter desktop applications.

## Projects

- `tkinter-widgets/miles_to_km.py` — minimal GUI converter; demonstrates grid layout and Entry/Label/Button wiring
- `tkinter-widgets/widget_examples.py` — reference sheet for common Tkinter widgets (Label, Button, Entry, Text, Spinbox, Scale, Checkbutton, Radiobutton, Listbox)
- `pomodoro-timer/` — full Pomodoro timer cycling through four focus sessions and breaks, with Windows audio alerts and a pause/resume button
- `password-manager/` — encrypted credential vault using PBKDF2-HMAC-SHA256 + Fernet; master-password protected with a 3-attempt lockout
